Overview :
This project is a Java console application that simulates a shopping system for a multi-category store. Users can browse and purchase items from categories such as Grocery, Snacks & Drinks, Beauty & Personal Care, Electronics, and Sports Items. The program calculates the total bill in real-time as the user selects items and enters quantities.

Features :
The application provides a main menu with all available categories. After selecting a category, users can view the items and their prices. Users can choose an item, enter the quantity they want to buy, and the total bill is updated immediately. Users have the option to continue shopping within the same category, go back to the main menu to explore other categories, or exit and generate the final bill.
When exiting, the program prompts the user to enter their name and mobile number. The mobile number is validated to ensure it has exactly 10 digits. After the input is validated, the program prints a detailed bill including the current date and time, the user’s name, mobile number, and the total amount payable, simulating a real shopping receipt.

Implementation Details :
The application demonstrates the use of loops and switch-case statements to navigate the menus and process user input. Methods are used to organize the code into manageable sections, showing basic object-oriented programming principles. The program also uses Java’s date and time API to display the current timestamp on the bill. Input validation ensures the program handles user errors gracefully, especially for mobile number entry.

Learning Outcomes :
This project helps in understanding how to create a menu-driven console application using Java. It covers concepts such as classes, methods, loops, conditional statements, input validation, and running calculations. The application also provides an opportunity to practice structuring code in a readable and maintainable way. The program can be further enhanced by adding features like discount calculation, storing transaction history, or implementing a graphical user interface.
